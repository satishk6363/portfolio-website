# Portfolio Website Deployment on AWS S3

This project is a personal portfolio website hosted using AWS S3 Static Website Hosting.

## Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub
- AWS S3

## Project Workflow

VS Code → Git → GitHub → AWS S3 Deployment

## Features

- Responsive portfolio website
- Static website hosting using AWS S3
- Version control using Git
- Source code managed on GitHub

## Deployment Steps

### 1. Open Project Folder

```bash
cd portfolio-website 
2. Initialize Git
git init
3. Add and Commit Files
git add .
git commit -m "Initial commit"
4. Push Code to GitHub
git push origin main
git push origin main
5. AWS S3 Static Website Hosting
Create S3 bucket
Upload website files
Enable static website hosting
Configure bucket policy
Access website using S3 endpoint
